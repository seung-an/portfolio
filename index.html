Project

VEXEL(동영상 스트리밍 사이트)

동영상을 자유롭게 업로드하고 관리 할 수 있으며 시청 할 수 있는 스트리밍 웹사이트 입니다.
출퇴근 또는 여가시간에 동영상 시청을 하는 사람들이 많아 졌고 이처럼 실생활에서 많이 사용하는 웹사이트를 개발해보자 라는 마음으로 개발하게 되었습니다.

Spring security + JWT토큰을 이용하여 사용자 인증을 구현하는 부분과 해당 토큰들을 어디에 저장하는지에 대해 깊게 생각 해 볼수있어 의미있는 경험이었습니다.

JPA를 이용하여 직접 SQL문을 작성하지 않고 데이터를 관리 하여 기존에 MyBatis로 연동하는 것과는 다른 방식을 시도 해 볼 수 있었습니다.

AWS EC2를 이용하여 서버에 배포하는 과정을 경험 할 수 있었고 업로드한 동영상을 프로젝트파일에 저장하는것이 아니라 Amazon S3클라우드에 저장하는 과정도 경험 할 수 있었습니다.
